PWM - DESCRIPTION (wire #)
0 - LEFT DRIVE MTR
1 - RIGHT DRIVE MTR
2 - BOTTOM SHOOTER MTR
3 - TOP SHOOTER MTR
4 - BEATER BAR MTR
5 - OUTER ACQUISITION MTR
6 - LEFT HANGING ARM MTR
7 - RIGHT HANGING ARM MTR
8 - LEFT DEFENSE ARM MTR
9 - RIGHT DEFENSE ARM MTR


DIO - DESCRIPTION (wire #)
0 - HALL EFFECT SENSOR
1 - LEFT DRIVE ENCODER:A
2 - LEFT DRIVE ENCODER:B
3 - RIGHT DRIVE ENCODER:A
4 - RIGHT DRIVE ENCODER:B
5 - BALL DETECT PHOTOEYE
6 - LEFT HANGING ARM LIMIT
7 - RIGHT HANGING ARM LIMIT
8 - LEFT DEFENSE ARM LIMIT
9 - RIGHT DEFENSE ARM LIMIT
MXP0 - LEFT DEFENSE ARM ENCODER:A
MXP1 - LEFT DEFENSE ARM ENCODER:B
MXP2 - RIGHT DEFENSE ARM ENCODER:A
MXP3 - RIGHT DEFENSE ARM ENCODER:B


RELAY - DESCRIPTION (wire #)
0 - DRIVE GEAR SHIFT SOLENOID
1 - DRIVE GEAR SHIFT SOLENOID
2 - PTO SHIFT SOLENOID
3 - HANGING PIN SOLENOID

AI - DESCRIPTION (wire #)
0 - GYRO SENSOR
1 - ULTRASONIC SENSOR
